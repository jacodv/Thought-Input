# Installer assets

These files are generated by CI (or the build) from sources in `windows-app/src/ThoughtInput/Resources/`:

| File | Source | How |
| --- | --- | --- |
| `AppIcon.ico` | `src/ThoughtInput/Resources/AppIcon.png` | `magick AppIcon.png -define icon:auto-resize=256,128,64,48,32,16 AppIcon.ico` |
| `WizardLarge.bmp` | `assets/branding/logo-mark.svg` | 164x314 BMP, generated in CI |
| `WizardSmall.bmp` | `assets/branding/logo-mark.svg` | 55x58 BMP, generated in CI |

Inno Setup needs `.bmp` (24-bit) for wizard imagery. CI generates them with ImageMagick.
The script tolerates missing wizard images gracefully (`WizardImageStretch=no` keeps any size).
